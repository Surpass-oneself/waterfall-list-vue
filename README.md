# waterfall-list-vue

- 非传统式瀑布流`demo`

- 当前项目`node.js`版本要求`18.x`或更高版本

[预览地址](https://huangjingsheng.gitee.io/hjs/waterfall/#/better)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
